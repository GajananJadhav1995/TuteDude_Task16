Student App

This is a basic project where I display student data using HTML, CSS and JavaScript.

When the page loads it shows all students.

There is a search box and button. When I enter a name:
- If I type starting letter like S, it shows students starting with S
- If I type full name, it shows exact match

How to run:
Open index.html in browser.
